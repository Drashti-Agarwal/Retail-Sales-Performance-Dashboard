# Retail Sales Performance Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to analyze retail sales performance. It provides insights into sales, profit, customers, transactions, and product performance.

## Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Customers
- Total Transactions
- Profit Margin
- Monthly Sales Trend
- Sales by Product Category
- Sales by Age Group
- Sales by Gender
- Interactive Slicers

## Tools Used

- Power BI
- DAX
- Power Query
- Excel

## Dataset

Retail Sales Dataset containing sales, customer, product, and transaction information.
